FastApi: It is super fast python web framework. Very modern and nice feature of scynchronous programming.

Features:
1. Automatic Docs
2. Provide Swagger UI
3. ReDoc UI
4. Use pydantic library
5. Based on Open Api and JSON Schema
6. Code editor auto compelete in vs code
7. Security and Authenmtication
8. Dependency Injection and Unlimited plugin tested
9. Use starlette features
10. SQL, NoSQL and GrapQL Supported
